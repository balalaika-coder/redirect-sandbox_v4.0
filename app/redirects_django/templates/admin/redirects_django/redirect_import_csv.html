Import Redirects from CSV

Upload a CSV file with the following columns:

  • site_slug — required
  • source — required (must start with /)
  • destination — required
  • status_code — optional (default: 301)
  • match_type — optional (default: exact)
  • geo_aware — optional (true/false, default: false)
  • priority — optional (default: 0)
  • utm_source, utm_medium, utm_campaign — optional
  • notes — optional

Note: This tool uses upsert behavior. If a redirect with the same site + source already exists, it will be updated. Otherwise, a new redirect is created.
